Principais Bibliotecas em Java

Java é uma linguagem rica em bibliotecas que facilitam o desenvolvimento de diversas aplicações. Aqui estão algumas das bibliotecas mais populares:

1. JDBC (Java Database Connectivity)

A biblioteca JDBC permite que você conecte seu aplicativo Java a um banco de dados relacional. Com ela, você pode realizar operações como inserção, exclusão, atualização e consulta a dados.

import java.sql.*;

public class ConexaoBanco {
    public static void main(String[] args) {
        try {
            Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/meubanco", "usuario", "senha");
            Statement stmt = conn.createStatement();
            ResultSet rs = stmt.executeQuery("SELECT * FROM usuarios");
            
            while (rs.next()) {
                System.out.println(rs.getString("nome"));
            }
            conn.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

2. JUnit

JUnit é uma biblioteca usada para escrever e executar testes automatizados em Java. Ela é muito importante no processo de desenvolvimento orientado a testes (TDD).

import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.assertEquals;

public class CalculadoraTest {
    
    @Test
    public void testSoma() {
        Calculadora calc = new Calculadora();
        assertEquals(5, calc.somar(2, 3));
    }
}

3. Apache Commons

A Apache Commons oferece várias bibliotecas úteis para tarefas comuns de programação, como manipulação de coleções, entrada/saída de arquivos, manipulação de strings, entre outras.

import org.apache.commons.lang3.StringUtils;

public class ExemploApache {
    public static void main(String[] args) {
        String texto = "  Java é incrível!  ";
        System.out.println(StringUtils.strip(texto));  // Remove os espaços extras
    }
}

4. Gson

Gson é uma biblioteca desenvolvida pelo Google para converter objetos Java em formato JSON e vice-versa, facilitando a manipulação de dados em APIs web.

import com.google.gson.Gson;

public class ExemploGson {
    public static void main(String[] args) {
        Pessoa pessoa = new Pessoa("João", 30);
        Gson gson = new Gson();
        String json = gson.toJson(pessoa);
        System.out.println(json);
    }
}

class Pessoa {
    private String nome;
    private int idade;

    public Pessoa(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
    }
}

5. Log4j

Log4j é uma biblioteca para registro de logs em Java, útil para rastrear eventos, erros e atividades do sistema.

import org.apache.logging.log4j.LogManager;
import org.apache.logging.log4j.Logger;

public class ExemploLog {
    private static final Logger logger = LogManager.getLogger(ExemploLog.class);

    public static void main(String[] args) {
        logger.info("Informação de log");
        logger.error("Erro de log");
    }
}

Sintaxe Básica de Java

Aprenda os fundamentos da linguagem Java com exemplos de código simples e práticos.

1. Estrutura de um Programa

public class Main {
    public static void main(String[] args) {
        System.out.println("Olá, Mundo!"); // Exibe uma mensagem no console
    }
}

Resultado: Quando você executa esse programa, o Java irá imprimir "Olá, Mundo!" na saída do console. Esse é o exemplo clássico de um programa de início, utilizado para testar se o ambiente de desenvolvimento está configurado corretamente.

2. Declaração de Variáveis

int idade = 25;             // Inteiro
double altura = 1.75;       // Número decimal
char inicial = 'J';         // Um único caractere
String nome = "João";       // Texto (String)
boolean ativo = true;       // Verdadeiro ou falso

Resultado: Nesse exemplo, declaramos diferentes tipos de variáveis em Java: int para números inteiros, double para números decimais, char para um único caractere, String para texto e boolean para valores lógicos (verdadeiro ou falso). Essas variáveis podem ser usadas para armazenar dados em seu programa.

3. Controle de Fluxo

Condicional (if/else):

int idade = 18;
if (idade >= 18) {
    System.out.println("Você é maior de idade.");
} else {
    System.out.println("Você é menor de idade.");
}

Resultado: O programa verifica se a variável idade é maior ou igual a 18. Se for, imprime "Você é maior de idade."; caso contrário, imprime "Você é menor de idade.".

Loop (for):

for (int i = 0; i < 5; i++) {
    System.out.println("Contagem: " + i);
}

Resultado: O loop for vai começar com i = 0 e vai incrementar i até atingir 5 (excluindo 5). A cada iteração, o programa imprime "Contagem: " seguido do valor de i.

Loop (while):

int contador = 0;
while (contador < 5) {
    System.out.println("Contador: " + contador);
    contador++;
}

Resultado: O loop while funciona de maneira semelhante ao for, mas a verificação da condição ocorre antes de cada iteração. O programa vai imprimir "Contador: 0", "Contador: 1", até "Contador: 4" e, depois disso, o loop será interrompido quando contador atingir 5.
